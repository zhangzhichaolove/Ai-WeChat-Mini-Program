## 项目简介
这是一个基于微信小程序开发的社交应用，包含朋友圈、个人主页、消息中心等功能模块。

## 功能特性
- 朋友圈：用户可以浏览和发布动态
- 个人主页：展示用户个人信息和相关内容
- 消息中心：处理用户消息通知
- 设置中心：用户可以管理个人设置
